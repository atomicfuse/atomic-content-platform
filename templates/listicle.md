# Listicle Article Template

## Structure
- Catchy numbered title (e.g., "10 Best...", "7 Must-Try...")
- Brief intro paragraph (2-3 sentences) setting context
- Numbered items, each with:
  - Item heading (## N. Item Name)
  - 2-3 paragraphs of description
  - Why it matters / key takeaway
- Conclusion paragraph summarizing the list

## Tone
- Engaging and opinionated
- Each item should feel like a mini-review
- Use specific details, names, and facts

## Length
- 800-1200 words
- 5-15 items depending on depth

## SEO
- Title includes the number and primary keyword
- Description summarizes the list theme
- Tags should include the category and specific items mentioned
